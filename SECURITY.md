# Security Policy

## Supported Versions

The `main` branch is the supported line for security updates.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use one of these channels:

1. GitHub Security Advisories (preferred):
   - https://github.com/sithukyaw007/local-ai-workload/security/advisories/new
2. If advisory flow is unavailable, open a Discussion without exploit details and request private contact:
   - https://github.com/sithukyaw007/local-ai-workload/discussions

## Response Targets

- Initial acknowledgment: within 72 hours.
- Initial triage and severity assessment: within 7 days.
- Fix timeline: depends on severity and complexity, with status updates provided.

## Disclosure Process

- We will validate and reproduce the report.
- We will prepare and test a fix.
- We will publish a coordinated disclosure with mitigation guidance.
